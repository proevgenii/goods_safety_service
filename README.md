# goods_safety_service

## Сервис картины рынка безопасных товаров


Сборка сервиса:

```bash
docker-compose build
```

Запуск сервиса:

```bash
docker-compose up -d
```

Остановка сервиса:

```bash
docker-compose down
```

## Локальный сервис (работает после запуска)
[Ссылка](http://localhost:8000/)
